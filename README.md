# VaultRAG

Enterprise RAG platform with hybrid search and citation tracking.

## About
VaultRAG is a Retrieval-Augmented Generation platform designed for enterprise use cases. It combines hybrid search strategies with built-in citation tracking to deliver accurate, sourced AI responses from your organization's knowledge base.

## Tech Stack
- Python, FastAPI
- Next.js, TypeScript, Tailwind CSS
- Supabase (PostgreSQL + pgvector)
- OpenAI, Cohere, sentence-transformers
- Celery, Redis

## Status
Active development - core architecture and project structure established.
